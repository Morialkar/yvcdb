# Phase 3 — Fondations

Implémente uniquement les tâches de fondation approuvées dans `AFTER_PLAN.md`. Lis et respecte tous les documents AFTER avant toute modification.

Crée la structure minimale, les manifestes, l'outillage qualité, le banc de tests et la base CI prévus par l'architecture. N'implémente pas les fonctionnalités réservées aux tâches suivantes. Documente chaque dépendance ajoutée.

Écris les tests avec chaque unité de logique : cas nominal, au moins un cas limite et au moins un cas d'erreur. Marque toute inférence inévitable `ASSUMPTION:` et tout code sensible `REQUIRES_REVIEW:`. Une hypothèse ne peut jamais contredire une contrainte approuvée.

Lance formatage, analyse statique, build et tests disponibles. Rapporte fichiers modifiés, décisions, hypothèses, marqueurs de revue, commandes et critères d'acceptation.
